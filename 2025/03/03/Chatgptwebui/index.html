怎么搭建一个自己的ChatGPT

发表于 2025-03-03 更新于 2025-03-15 分类于 网站

之前不是自己搭了个ChatGPT的web么，现在需要重新搭一次，干脆写成一篇文章记录一下。
借鉴了一下Fomalhaut大佬的文章

前置条件

  1. 需要一个 OPENAI 账号(chatGPT 账号)，或者有效的 API KEY，没有的话叫朋友借一个 key
  2. 需要一个国内的域名，没有的可以去硅云白嫖一个
  3. 需要有能够访问外国网站的能力(你懂的)
